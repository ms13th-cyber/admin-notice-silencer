管理画面に乱立するプラグインの通知やバナーをワンクリックで永続的にミュートし、作業に集中できるクリーンな環境を提供する軽量プラグインです。

== Description ==

Admin Notice Silencer は、WordPress 管理画面の利便性を向上させるための、単一機能（Single-function）に特化した軽量プラグインです。

多くのプラグインを導入すると、管理画面の上部が「アップデートのお知らせ」や「レビューのお願い」などのバナーで埋め尽くされ、本来の作業領域が狭くなってしまいます。本プラグインは、それらの通知をユーザーごとに個別に非表示化（ミュート）し、必要な時だけ再表示できる仕組みを提供します。

主な特徴：

* **ワンクリック・ミュート**: 各通知の右側に配置された「ミュート」ボタンから、即座に非表示化が可能です。
* **プラグイン単位の識別**: 通知のCSSクラス等を解析し、特定のプラグインから出力される通知を系統立ててブロックします。
* **高度な安全弁**: 致命的なシステムエラー（notice-error）やコアの更新通知（update-nag）はミュート対象外として保護され、運用の安全を確保します。
* **ユーザーごとの個別設定**: ミュート状態はログインユーザーごとに保存されるため、他の管理者の表示には影響を与えません。
* **統合管理画面**: 設定メニュー内に専用ページを提供。現在ミュート中の項目を一覧確認し、必要に応じて個別に解除できます。
* **超軽量設計**: JavaScript と最小限の CSS 注入のみで動作し、サイトのパフォーマンスに影響を与えません。

== Installation ==

1. プラグインフォルダを以下へアップロードします。
   `/wp-content/plugins/admin-notice-silencer`

2. WordPress管理画面の「プラグイン」から有効化します。

3. 管理画面の上部に表示される通知バナーの右側にある「ミュート」ボタンをクリックします。

4. ミュートした通知を再表示したい場合は、管理画面の「設定 > Notice Silencer」から解除を行ってください。

== Usage ==

このプラグインは有効化するだけで機能します。複雑な設定は不要です。

* **ミュートする**: 通知の右端（閉じるボタンの左隣）に表示される「ミュート」ボタンを押すと、その通知とそれに関連するプラグインの通知が非表示になります。
* **管理と解除**: 「設定」メニュー内の「Notice Silencer」ページを開くと、現在ミュートしている通知の識別IDが一覧表示されます。右側の「解除」ボタンを押すと、その通知が再び表示されるようになります。
